# 立场与偏见分析

> 这份文档处理的不是“来源够不够权威”，而是“即使材料真实存在，它的叙事是否带有明显立场、利益过滤或视角缺口”。
>
> 当分析对象是媒体报道、机构声明、商业宣传、政策表述、争议评论、带强烈价值判断的总结时，应在常规批判性评估之外，补做这一步。

## 1. 边界

这份文档关注：

- 立场倾向
- 利益关系
- 叙事框架
- 缺失视角
- 偏见修正

这份文档不负责：

- 来源等级划分
- 时效性验证
- 事实核验
- 数据完整性审查
- 引用可溯源性

这些问题属于 `source-quality-control` 的范围。

## 2. 何时启动

出现以下任一情况时，默认启动立场与偏见分析：

- 话题本身存在明显争议、阵营分化或价值冲突
- 文本目标明显带有说服、动员、营销、辩护、归责或站队意图
- 发布者是媒体、企业、机构、研究组织、意见领袖或利益相关方
- 材料可能影响技术选型、商业判断、公共议题理解或风险认知
- 文本看起来“论证很顺”，但几乎没有处理反例、对立观点和代价

## 3. 分析目标

- 识别这段材料主要服务于谁的叙事
- 区分哪些是事实描述，哪些是立场包装后的判断
- 找出被忽略的反面视角、代价和利益受损方
- 让最终结论更平衡，而不是更会附和某一方

## 4. 六步检查框架

### 4.1 来源背景

先判断“谁在说”：

- 发布者是谁，处于什么组织位置
- 是否与话题存在直接利益关系
- 过去是否长期保持某种固定立场
- 当前面向谁说，想影响谁
- 为什么在这个时间点发布

重点不是做动机臆测，而是识别表达位置。位置不同，叙事选择通常也不同。

### 4.2 表达信号

检查“它是怎么说的”：

- 是否频繁使用绝对化表达，如“必然”“完全”“唯一”“永远”
- 是否大量使用情绪化、标签化或带羞辱色彩的词语
- 是否只展示支持自身结论的案例，而回避反例
- 是否把复杂问题压缩成二元对立
- 是否刻意淡化不确定性、条件限制和失败成本

这些信号不自动等于“错误”，但通常意味着文本带有较强的引导意图。

### 4.3 论证结构

检查“它为什么这么说”：

- 哪些部分是事实，哪些部分是解释，哪些部分是结论
- 事实与结论之间是否存在因果跳跃
- 是否处理了主要反例和反方论据
- 是否把局部案例外推成普遍结论
- 是否偷换了比较基准、评价口径或时间范围

如果一段文本只展示结论，不展示推理链，就要警惕它是“叙事优先”而不是“分析优先”。

### 4.4 缺失视角

检查“它没说什么”：

- 谁的视角没有被代表
- 哪些受损方、承担成本方或边缘群体被忽略
- 是否缺少关键时间线、历史背景或对立案例
- 是否存在至少一个合理替代解释没有被提到
- 当前叙事是否把复杂问题收缩成对自己有利的局部切面

很多偏见不是“说错了”，而是“故意或无意漏掉了关键部分”。

### 4.5 利益相关方地图

至少画出四类角色：

- 发布方：谁在发声
- 受益方：谁会因该叙事获益
- 受损方：谁可能承担成本或被排除
- 观察方：谁相对中立，或拥有不同解释框架

如果条件允许，再补一层：

- 主流观点
- 对立观点
- 中立观点
- 利益相关方观点

这一步的目标，是避免把“声音最大的一方”误当成“事实最完整的一方”。

### 4.6 偏见修正

完成识别后，不要停在“看出有偏见”，还要做修正：

- 用更中性的语言重述原命题
- 补上最强的反方论据，而不是找最弱 strawman
- 明确哪些判断仍只是基于现有材料的推断
- 在证据不足时主动降低结论力度
- 指出若要提高判断质量，下一步最该补查什么

## 5. 常见偏见类型

### 5.1 确认偏误

- 只收集支持既有观点的信息
- 主动忽视或贬低相反证据
- 把模糊证据解释成对自己有利

### 5.2 锚定效应

- 被最早接触到的叙事框架锁定
- 后续信息只能围绕初始判断微调
- 明明出现新信息，结论却几乎不变

### 5.3 标签化与刻板印象

- 用简单标签替代复杂事实
- 把群体印象直接套到个体或事件上
- 通过给对方贴标签削弱其论点

### 5.4 利益偏见

- 立场与经济利益、组织目标或政治收益高度绑定
- 对自家代价轻描淡写，对他方代价放大呈现
- 把局部利益包装成普遍利益

### 5.5 阵营认同偏见

- 先按“我们/他们”划分，再决定怎么解释事实
- 把对立方的证据默认视为不可信
- 把本方叙事默认视为更高尚、更客观

## 6. 输出模板

可直接用于分析明显带立场的材料：

```text
立场识别：
- 当前材料主要倾向于 [...]

表达与论证信号：
- 明显信号包括 [...]

利益相关方：
- 发布方/受益方/受损方/被忽略方分别是 [...]

缺失视角：
- 当前材料没有处理 [...]

暂定判断：
- 基于现有材料，我认为它更像 [中性分析 / 单边叙事 / 利益表达 / 混合型文本]。
- 这是基于资料的推断，不等于完成事实核验。

平衡建议：
- 若要形成更可靠判断，下一步应补看 [...]
```

## 7. 避免的误区

- 不要把“有立场”直接等同于“全部无效”
- 不要为了显得平衡，就给弱证据与强证据同等权重
- 不要把立场分析替代事实核验，两者不是一回事
- 不要把偏见分析做成人身动机揣测或道德审判
- 不要只指出对方偏见，却不检查自己当前分析的立场惯性
