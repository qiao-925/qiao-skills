# 🚨 立场客观性警告Prompt

## 🎯 核心目标
**多角度验证 > 单一立场** - 在信息爆炸的时代，我们需要识别立场偏见，确保客观分析和多角度思考

## 📋 Prompt模板

### 🔍 立场分析模式

```
你是一位立场分析专家，擅长识别信息源中的立场偏见和潜在误导。

**分析框架**：
1. **来源背景**：信息发布者的身份、利益关系、历史立场
2. **语言特征**：情感化词汇、绝对化表达、选择性呈现
3. **论证逻辑**：论据完整性、反例处理、因果关系
4. **利益关联**：与特定群体、行业、观点的关联度

**输出要求**：
- 🎯 立场识别：[主要立场倾向]
- ⚠️ 偏见警告：[潜在偏见点]
- 🔍 缺失视角：[被忽略的反面观点]
- 📊 可信度评估：[1-10分，说明理由]
- 💡 平衡建议：[如何获得更全面的视角]

请分析以下内容：[粘贴内容]
```

### 🚨 客观性验证模式

```
作为客观性验证专家，请帮我：

**验证维度**：
- **事实vs观点**：哪些是客观事实，哪些是主观判断？
- **数据完整性**：是否提供了完整的数据背景？
- **对比分析**：是否有其他观点或数据作为对比？
- **时间维度**：信息是否过时，当前是否仍然有效？
- **利益冲突**：发布者是否有利益相关？

**输出格式**：
✅ 客观事实：[可验证的客观信息]
❓ 主观判断：[需要进一步验证的观点]
⚠️ 潜在偏见：[立场倾向和可能误导]
🔄 补充视角：[建议查阅的其他来源]

请对以下内容进行客观性验证：[粘贴内容]
```

### 🎯 多角度分析模式

```
你是一位多角度分析专家，擅长从不同立场分析同一问题。

**分析要求**：
1. **主流观点**：当前主流立场和论据
2. **对立观点**：相反立场的主要论据
3. **中立视角**：第三方或学术界的观点
4. **利益相关方**：各利益群体的立场分析
5. **历史背景**：问题的历史演变和背景

**输出结构**：
📊 **观点分布**：[各立场的支持度和影响力]
⚖️ **论据对比**：[各方论据的强弱分析]
🎯 **立场地图**：[各方的利益关系和立场关联]
💡 **平衡建议**：[如何形成客观判断]

请对以下话题进行多角度分析：[具体话题]
```

### 🔄 偏见纠正模式

```
我们正在分析一个可能有立场偏见的内容。

**当前内容**：[描述内容]

**纠正目标**：
- 识别潜在的立场偏见
- 补充缺失的相反观点
- 提供更平衡的视角
- 避免单一立场误导

**协作流程**：
1. 你识别立场偏见点
2. 我确认或补充背景信息
3. 你提供平衡视角
4. 我评估客观性提升
5. 迭代直到达到平衡

请开始第一轮立场分析。
```

## 🎯 使用场景

### 📰 新闻分析
- 识别媒体报道的立场倾向
- 对比不同媒体的报道角度
- 避免被单一媒体观点误导

### 📊 研究报告
- 分析研究机构的背景立场
- 识别研究方法的潜在偏见
- 对比不同研究结论

### 💼 商业信息
- 分析企业发布信息的利益动机
- 识别竞争对手的立场偏见
- 平衡不同利益相关方的观点

### 🎓 学术讨论
- 识别学术争议中的立场分歧
- 分析不同学派的理论立场
- 保持学术讨论的客观性

## 🔧 立场识别技巧

### 📍 语言特征识别
- **绝对化表达**："绝对"、"完全"、"永远"
- **情感化词汇**：过度褒贬的形容词
- **选择性呈现**：只展示支持观点的证据
- **标签化语言**：给对立观点贴标签

### 🎯 利益关系分析
- **资金来源**：谁为信息付费？
- **组织背景**：发布者的历史立场
- **利益冲突**：是否涉及经济利益？
- **政治关联**：与特定政治立场的关联

### 📊 数据完整性检查
- **样本代表性**：数据是否具有代表性？
- **时间范围**：是否选择了有利的时间段？
- **对比基准**：是否有合适的对比对象？
- **统计方法**：统计方法是否合理？

## 💡 客观性提升策略

### 🔍 多源验证
- 查阅不同立场的多个来源
- 对比官方和非官方信息
- 参考国际和国内不同视角
- 关注学术研究和媒体报道

### ⚖️ 平衡思考
- 主动寻找相反观点
- 考虑不同利益相关方
- 分析历史背景和演变
- 评估长期和短期影响

### 🎯 批判性思维
- 质疑信息来源的动机
- 分析论证逻辑的完整性
- 识别潜在的认知偏差
- 保持开放和怀疑的态度

## 🚨 常见偏见类型

### 🎯 确认偏误
- 只关注支持自己观点的信息
- 忽视或贬低相反证据
- 过度解释模糊信息

### 📊 锚定效应
- 被初始信息过度影响
- 难以调整后续判断
- 忽视新的相关信息

### 🏷️ 刻板印象
- 基于群体特征判断个体
- 忽视个体差异和复杂性
- 强化既有偏见

### 💰 利益偏见
- 因经济利益影响判断
- 忽视公共利益考虑
- 为特定群体辩护

---

*记住：在信息时代，最重要的不是知道什么，而是知道如何知道。我们的目标是成为信息的理性过滤器，而不是偏见的放大器。*
