# UI 自然语言编辑详细说明

## 1. 功能概述

使用自然语言描述 UI 调整，自动应用到代码。

## 2. 使用方式

### 2.1 命令格式

```
/browser-edit <自然语言描述>
```

### 2.2 示例

```
/browser-edit 调整聊天输入框圆角为 20px
/browser-edit 将侧边栏背景色改为浅灰色
/browser-edit 增加按钮间距为 10px
```

## 3. 工作流程

```
1. 用户发送自然语言描述
    ↓
2. 启动浏览器（如未启动）
    ↓
3. 定位目标元素
    ↓
4. 使用可视化编辑器调整
    ↓
5. 自动应用更改到代码
    ↓
6. 验证效果
```

## 4. 支持的调整类型

### 4.1 样式调整

| 类型 | 示例 |
|------|------|
| 颜色 | "改为蓝色"、"背景色 #f0f0f0" |
| 尺寸 | "宽度 300px"、"增加高度" |
| 间距 | "增加间距"、"padding 20px" |
| 圆角 | "圆角 10px"、"去掉圆角" |
| 边框 | "添加边框"、"边框颜色红色" |

### 4.2 布局调整

| 类型 | 示例 |
|------|------|
| 对齐 | "居中对齐"、"右对齐" |
| 排列 | "水平排列"、"垂直排列" |
| 间距 | "增加列间距" |

### 4.3 文本调整

| 类型 | 示例 |
|------|------|
| 字体大小 | "字体放大"、"14px" |
| 字体颜色 | "文字改为灰色" |
| 字体粗细 | "加粗" |

## 5. 脚本使用

### 5.1 脚本位置

`scripts/browser_edit.py`

### 5.2 参数

| 参数 | 说明 |
|------|------|
| `--description` | 自然语言描述 |
| `--element` | 目标元素选择器（可选） |
| `--preview` | 仅预览不应用 |

### 5.3 示例

```bash
python scripts/browser_edit.py --description "调整按钮圆角为 10px"
```

## 6. 注意事项

### 6.1 前提条件

- Streamlit 应用已启动
- 浏览器可访问应用

### 6.2 限制

- 只支持 CSS 样式调整
- 不支持添加/删除组件
- 复杂布局调整可能需要手动代码修改

### 6.3 最佳实践

1. 描述尽量具体
2. 一次只调整一个属性
3. 调整后验证效果
