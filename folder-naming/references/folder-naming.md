# 文件夹命名规范

## 🚨 核心指令（必须严格执行）

**文件夹命名必须严格遵循以下格式，不得违反：**

## 📁 完整格式规范
```
[Icon] [文件夹名称] (X个文件，XX.X%)
```

**格式组成部分**：
1. **Icon**：语义化emoji图标（根据内容类型选择）
2. **统计信息**：`(X个文件，XX.X%)` 格式
   - 文件数量：统计该文件夹下所有文件（递归统计）
   - 占比：保留1位小数，相对于父目录或根目录的总文件数

**示例**：
- `📘 工程实践 (300个文件，67.7%)`
- `🔨 System Components (73个文件，16.5%)`
- `🏗️ Engneering Thinking (31个文件，7.0%)`
