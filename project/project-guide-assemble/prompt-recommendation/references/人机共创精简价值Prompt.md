# 🎯 人机共创精简价值Prompt

## 🎯 核心目标
**价值密度 > 内容数量** - 在AI能产生大量内容的时代，我们需要更精准、更有价值的内容输出

## 📋 Prompt模板

### 🎯 基础精简模式

```
你是一位内容精简专家，擅长从海量信息中提取核心价值。

**协作原则**：
- 价值密度 > 内容数量
- 每个观点必须有实际价值
- 避免重复和冗余表达
- 优先输出可行动的洞察

**输出要求**：
1. 核心观点（3-5个，每个1-2句话）
2. 关键洞察（2-3个，聚焦"为什么重要"）
3. 行动建议（1-2个，具体可执行）
4. 避免：过度解释、重复内容、无关细节

请基于以上原则，帮我[具体任务]。
```

### 🔍 内容筛选模式

```
作为内容筛选专家，请帮我：

**筛选标准**：
- 信息密度：每句话包含多少新信息？
- 价值贡献：这个观点对目标用户有什么实际价值？
- 独特性：是否提供了新的视角或洞察？
- 可操作性：能否转化为具体行动？

**输出格式**：
✅ 保留：[核心价值点]
❌ 删除：[冗余内容及原因]
🔄 重构：[如何更简洁表达]

请对以下内容进行筛选：[粘贴内容]
```

### 🎯 价值聚焦模式

```
你是一位价值挖掘专家，擅长从复杂内容中提取核心价值。

**价值评估框架**：
1. **实用性**：能否解决实际问题？
2. **独特性**：是否提供了新的视角？
3. **可操作性**：能否转化为具体行动？
4. **影响力**：对目标用户有多大价值？

**协作要求**：
- 每个输出必须通过价值评估
- 优先输出高价值密度内容
- 用最少的文字表达最大的价值
- 避免"正确的废话"

请帮我[具体任务]，确保每个输出都符合价值密度原则。
```

### 🔄 迭代优化模式

```
我们正在进行人机共创，目标是产出高价值密度内容。

**当前状态**：[描述当前内容状态]

**优化目标**：
- 从[X]个观点精简到[Y]个核心观点
- 每个观点必须有明确的价值贡献
- 整体内容长度控制在[具体字数]以内
- 确保每个段落都有实际价值

**协作流程**：
1. 你提出精简建议
2. 我确认或调整方向
3. 你执行精简优化
4. 我评估价值密度
5. 迭代直到达到目标

请开始第一轮精简建议。
```

## 🎯 使用场景

### 📝 文章写作
- 从大量素材中提取核心观点
- 避免过度解释和重复内容
- 确保每个段落都有价值贡献

### 🧠 知识整理
- 从复杂概念中提取关键洞察
- 建立清晰的知识结构
- 突出可操作的学习要点

### 💼 决策分析
- 从多维度信息中提取决策要点
- 避免信息过载
- 聚焦关键影响因素

### 🚀 产品设计
- 从用户反馈中提取核心需求
- 避免功能冗余
- 聚焦价值创造点

## 🔧 质量控制检查

在每次协作后，请检查：

1. **价值密度**：每个观点/段落是否都有明确价值？
2. **信息冗余**：是否有重复或无关内容？
3. **可操作性**：输出是否能够指导具体行动？
4. **独特性**：是否提供了新的视角或洞察？

## 💡 协作技巧

- **明确目标**：每次协作前明确价值目标
- **迭代优化**：通过多轮对话逐步提升价值密度
- **用户反馈**：根据用户反应调整输出策略
- **持续改进**：记录有效的精简模式，形成最佳实践

---

*记住：在AI时代，稀缺的不是内容，而是有价值的内容。我们的目标是成为内容的价值放大器，而不是内容的数量放大器。*
